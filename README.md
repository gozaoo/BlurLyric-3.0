# BlurLyric 3.0
<p>全新的使用tauri框架的美观多功能播放器！</p>

## 特性 (持续更新中)
- 1.✅ 使用vue、tauri等框架带来的轻量级播放器。
- 2.❌ 支持本地音乐播放、网易云账号、webdev云盘关联等等。
- 3.❌ 支持本地音乐tag自动补全
- 4.✅ 丝滑流畅动效体验
- 5.❌ 多设备、尺寸适配
- 6.❌ 串流播放
